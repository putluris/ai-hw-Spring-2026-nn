MNIST Recognition by Neural Network

This project implements a Convolutional Neural Network (CNN) to classify handwritten digits using the MNIST dataset.

Dataset:
The MNIST dataset was used, consisting of 60,000 training images and 10,000 test images of handwritten digits (0–9).

Model:
Convolutional Neural Network (CNN)

Training:
- Trained on MNIST training dataset
- Tested on MNIST test dataset
- 5 epochs
- Used data augmentation (RandomRotation)

Results:
Test Accuracy: 99.10%
